# Public Boundary

This repository contains the public static Akalynth website only.

It may contain:

- Static HTML, CSS, and JavaScript for the public site
- Public-safe screenshots
- Public lore and preview copy
- Preview shop, account, and housing pages
- Preview forum identity-seal policy and browser-local public-key fingerprints
- Public-safe documentation for this static site

It does not contain:

- The Akalynth game server
- Live economy authority
- Real account creation
- Real house ownership settlement
- Payment processing
- Operator credentials
- Runtime state
- Receipt authority
- PGP private keys or live signature verification authority
- Anti-cheat enforcement logic
- Private roadmap or planning material

All shop, account, and housing interactions in this repository are preview-only
unless explicitly connected to a named server receipt and verifier path in a
future release.

Forum Identity Seal copy is also preview-only. A local public-key fingerprint
can demonstrate the intended policy surface, but it does not verify detached
signatures, prove trust, or create live account authority.
