# Pre-Alpha Notice

Akalynth is in pre-alpha.

This repository is a static public preview. It is not a production service and
does not create real accounts, process real payments, grant item entitlements,
settle house ownership, run auctions, or write game-state receipts.

The website uses browser-local storage for preview character, cart, premium,
housing, forum, and forum identity-seal state. Clearing browser storage removes
that preview state.

Any future live feature must be backed by the appropriate server authority,
receipt, verifier, and release proof before public copy may describe it as live.

The forum Identity Seal is a static preview of public-key identity policy. It
can store a local PGP public-key fingerprint, but it does not verify signatures
or prove that a person, post, marketplace claim, guild claim, or governance
claim is true.
