# Contributing

This public repository is not currently accepting unsolicited code
contributions.

Pull requests may be closed without review unless they were requested or
pre-authorized by Akalynth maintainers.

## License Boundary

Opening a pull request does not grant any license to use Akalynth assets, code,
marks, lore, screenshots, game content, or website content beyond the rights
expressly granted in `LICENSE`.

This repository is public for inspection and preview under the current license.
Do not describe it as carrying public reuse rights unless the license changes in
a separate, explicit decision.

## Allowed Maintainer-Requested Changes

Maintainer-requested changes should stay within the public static site boundary:

- Static HTML, CSS, and JavaScript for the public site
- Public-safe screenshots and assets
- Public preview copy
- Repository governance documents and templates

Do not add:

- Server code
- Android code
- Secrets, tokens, keys, or credentials
- Payment configuration
- Deployment configuration
- Runtime logs or state
- Private roadmap material
- Anti-cheat internals
- Receipt authority or verifier internals
- Live account, payment, auction, or ownership claims without named production proof

## Security Reports

Do not open public issues or pull requests for vulnerabilities. Follow
`SECURITY.md`.
