Install via F-Droid

Add the Akalynth F-Droid repository once and your device keeps the client up to date automatically. Every build is signed by the same key, so each update is verified end to end.

  1. Install the F-Droid app (skip if you already have it).
  2. Add the Akalynth repo — on your phone tap Add Akalynth repo below, or in F-Droid open Settings → Repositories → + and scan the QR or paste the URL.
  3. Open the repo and install Akalynth. New beta builds arrive as updates.

Or sideload the APK

Prefer a one-off install? Download the latest signed build directly. You may need to allow installs from unknown sources on your device.
